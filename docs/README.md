## ğŸš€ Quick Start

### Prerequisites
```bash
# Required
- Node.js 18+
- Python 3.11+
- PostgreSQL 14+
- Redis 7+
- Docker & Docker Compose

# Optional
- Kubernetes cluster
- AWS account
- Stripe account
```

### Installation

1. **Clone Repository**
```bash
git clone https://github.com/yourorg/globalconnect.git
cd globalconnect
```

2. **Setup Environment**
```bash
# Frontend
cp .env.example .env.local
# Edit .env.local with your configuration

# Backend services
cd backend/event-lifecycle-service
cp .env.example .env
# Edit .env with your configuration
```

3. **Install Dependencies**
```bash
# Frontend
npm install

# Backend (each service)
cd backend/event-lifecycle-service
pip install -r requirements.txt
```

4. **Start Development Servers**
```bash
# Using Docker Compose (recommended)
docker-compose up -d

# Or manually
npm run dev                    # Frontend
npm run dev:gateway            # Apollo Gateway
npm run dev:realtime          # Real-Time Service
python manage.py runserver    # Event Lifecycle Service
```

5. **Access Application**
- Frontend: http://localhost:3000
- API Gateway: http://localhost:4000/graphql
- Real-Time: http://localhost:3002
- Admin Panel: http://localhost:3000/admin
